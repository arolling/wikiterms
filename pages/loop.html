Loops allow you to iterate a piece of code multiple times, with a series of arguments.

Sample Code:

for(var i = 1; i < 5; i ++) {
  alert("loop " + i);
}

Explanation:

In the above example the loop is initiated by the statement "for". Each time it runs, it increments the parameter "i" and brings up an alert, then runs again. This continues until "i" reaches 5, then it stops.
